# PolyGen
Un générateur d'images 2D low poly automatisé, permettant de créer des œuvres d'art algorithmiques sans compétences en dessin traditionnel.

## Objectifs

- Génération procédurale d'images low poly abstraites

- Conversion d'images existantes en style low poly

- Interface utilisateur simple pour contrôler les paramètres

- Export en différents formats
